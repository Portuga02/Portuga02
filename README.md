### 🚀 Sobre Mim

Sou um **Engenheiro de Software & Especialista em Desenvolvimento Web** focado em construir soluções robustas e escaláveis. Com uma base sólida no ecossistema PHP e JavaScript, atuo no desenvolvimento de aplicações Fullstack, priorizando boas práticas de arquitetura, performance e código limpo. Atualmente em transição ativa para o ecossistema **Java/Spring Boot**, aplicando os mesmos princípios de arquitetura em novos projetos.

- 👨‍💻 Atualmente focado em **Arquitetura de Software (SOLID, Clean Code, DTO, Clean Architecture)**.
- 🎓 Pós-graduado em Engenharia de Software, Banco de Dados e Desenvolvimento Web.
- 🛠️ Experiência em modernização de sistemas legados e criação de APIs RESTful.
- 🌱 Evoluindo em **Java + Spring Boot** e **Ionic/Angular** (mobile & frontend).
